## AES Experiments

This is a simple repo of mine to experiment with AES-256 encryption

## Building
```bash
# 1. Clone the repository
git clone --recursive https://github.com/hellisabove/aes-exp.git
cd aes-exp
# 2. Run the make command
make
```
